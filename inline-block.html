Inline/Block Elements

Inline Vs. Block

An inline element contains content whereas a block element is contained within it's container.

Inline Examples:

  • <span>
  • <a>
  • <img>

Block Examples:

  • <div>
  • <h1> - <h6>
  • <p>
  • <form>
  • <header>
  • <footer>
  • <section>
